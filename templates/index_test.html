01. AWS 시작하기

파이썬 Flask를 통한 AWS 경험하기

2019.11
hyeonazzz
Flask 를 통한 기능 구현을 배포하기 위해 로컬호스트(Localhost)가 아닌 AWS EC2를 활용하기

프론트엔드와의 통신을 위해 주소를 가지는 서버가 필요합니다.
AWS (Amazon Web Server)에 가입하여 관리 콘솔로 들어갑니다.

AWS Management Console


EC2 인스턴스 생성

개방성과 관리측면의 장점을 고려해 리눅스 우분투 서버를 추천합니다.

이후 General purpose의 프리 티어(Free Tier)로 시작하세요.

별도의 설정 없이 인스턴스 [시작]을 누르면 키 페어 설정 페이지가 나옵니다.

KeyPair001 등의 이름을 정해주고 키 페어 다운로드 후 [인스턴스 시작]을 누릅니다.


KeyPair 권한 설정

일반적으로 인스턴스 접속은 SSH 명령을 사용합니다.

접속 이전에 키 페어의 권한 설정 변경이 필요합니다.

  • Windows

Windows 환경인 경우 다운로드 받은 키 페어의 [속성] -> [보안] -> [고급]에 접근하여
Administrator와 SYSTEM을 제외한 사용자의 상속사용을 해제하고 권한을 제거하면 됩니다.

  • Linux

리눅스의 경우 터미널에서 아래 명령을 입력해 주시면 간단히 적용됩니다.

chmod 400 KeyPair001.pem

EC2 연결 및 실행

이제 CMD(명령 프롬프트)를 관리자 권한으로 실행하고 키페어 경로로 이동하세요.
EC2의 [연결]탭에 보이는 SSH 명령어를 입력해 줍니다. 

cd KeyPair가 저장된 경로
                
 ssh -i "KeyPair001.pem" ubuntu@ec2-52-79-163-195.ap-northeast-2.compute.amazonaws.com
